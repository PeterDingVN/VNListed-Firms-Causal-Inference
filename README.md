# Financial metrics prediction - Vietnamese listed firms

_____________________________________________________________

## Scope of work
_____________________________________________________________
### Data
This project uses data from FiinProX, of around 2000 listed firms from all 
platforms (HNX, HOSE, UPCoM) and additional sources (OTC, PRIVATE) in Vietnam.
The span of data is from 2015 to 2024.
The meaning of used variable can be found inside folder:

_**data/data_raw/var_definition.txt**_

### Goal
This project aims to predict 5 metrics: Revenue, EBITDA, Value-added, ROA, ROE.
And, the built model focus on predicting one timestep ahead, using multiple years
in the past.
